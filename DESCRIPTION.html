Allow you to generate variable named __all__ and reformat its value.

提供生成及格式化 __all__ 的能力。

Generation／生成

Menu:
  • Code → Generate... → __all__
  • Code → Generate... → __all__ (with imports)
菜单：
  • 代码 → 生成... → __all__
  • 代码 → 生成... → __all__（包含导入）

Reformatting／格式化

Menu:
  • Code → Optimize __all__
菜单：
  • 代码 → 优化 __all__
Shortcut／快捷键：
  • Windows｜Ctrl + Shift + D
  • Mac OS｜Control + Meta + D
